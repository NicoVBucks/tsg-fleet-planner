# TSG Fleet Planner

An unofficial Star Citizen community tool for mission leads — plan fleet composition, assign crew roles, evaluate readiness, and share summaries with your team.

**Made by the Community · Created by Daygonn · v0.9**

---

## Features

- **Fleet Builder** — Add ships by class (light fighter, medium fighter, heavy fighter, gunship, capital ship) and configure each one
- **Crew Assignment** — Assign player names to every slot, flag Trench Runners and FPS Operators
- **Score & Readiness** — Automatic analysis of your fleet composition across all mission phases (Idris Defense → Relay Assault → Trench Run → FPS Rescue → Vanduul Mauler)
- **Mission Timer** — Phase-by-phase stopwatch to track your operation in real time
- **Crew Summary Export** — Generate a shareable PNG or copy it to clipboard directly from the browser
- **Local Autosave** — Fleet and timer persist across page closes via `localStorage`
- **7 Languages** — English, Français, Deutsch, Italiano, Español, 日本語 + more

## Privacy

No account. No server. No analytics.

All data stays in your browser. Nothing is sent anywhere unless you export or share it yourself.

## Tech Stack

Pure HTML / CSS / JavaScript — no build step, no framework, no dependencies beyond web fonts and icon set loaded from CDN.

## Usage

Open `index.html` in any modern browser. No installation required.
